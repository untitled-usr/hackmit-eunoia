Frontpage assets (canonical)

Directory convention:

- `backgrounds/`: background images
  - `home-2.jpg`
- `icons/`: non-navigation icons
  - `home-shell.png`
- `icons/nav/`: sidebar navigation icons
  - `robot.png` (Agent)
  - `notebook.png` (Diary)
  - `speech-balloon.png` (Message)
  - `bottle.png` (Bottle)
  - `house.png` (Home)

Notes:

- Runtime URL base is `/assets/frontpage/...`
- Keep filenames lowercase with hyphen style.
